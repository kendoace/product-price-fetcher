=== Usage ===

Upload .csv file with comma delimited data: 
- the "URL" where the product price whould be fetched from 
- the "SKU" of the product that needs to be updated
for e.g.
url,sku
https://www.myurl.com/product-one/,123456789
https://www.myurl.com/product-two/,987654321
